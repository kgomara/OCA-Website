iOS 12 & Swift 4 Course

The course includes 50+ hours of HD video tutorials including ARKit 2 for making Augmented Reality apps as well as CoreML2 & CreateML.

AppleWatch Programming Course

The course covers tables, notifications, complications, syncing data between Phone and watch, timers, and much more.

AWS Certified Solutions Architect Course

This course starts with a broad overview of the AWS platform and then deep dives into the individual elements of the AWS platform, including Route53, EC2, S3, Cloud Front, Autoscaling, Load Balancing, RDS, RedShift, DynamoDB, EMR, VPC etc. Note - coursework only, I did not sit for the Certification exam.

AWS Certified Developer Associate Course

This course provides an understanding of core AWS services, uses, and basic AWS architecture best practices in order to become proficient in developing, deploying, and debugging cloud-based applications using AWS. Note - coursework only, I did not sit for the Certification exam.

Pragmatic MobX in React Course

This course starts with setting up MobX in React, connecting it to a view-layer (such as React), to performing asynchronous requests using MobX as state layer.

React Native Course

This course covers the theory behind React Native, its core concepts, how to build responsive designs that work on different device sizes, how to animate React Native apps, how to navigate around, use maps and the camera and much more.
